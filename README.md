## About Project

Simple application where users manage invoice.

## Requirement

- laravel v10.28.0
- php v8.2.11
- composer v2.5.8
- npm 10.2.0
- vue 3.3.6

## How To Run

- Clone this project
- Create database name: invoice-app-db
- Run : composer install
- Run : php artisan migrate
- Run : php artisan db:seed
- Run : php artisan serve
